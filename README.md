## Hey, I'm Krish

I am a developer from Nepal who likes building things that are practical, clean, and a little bit bold. My main world right now is **Python development**, **Machine Learning**, and **AI Engineering**, but I also enjoy creative automation, web interfaces, and design work that makes projects feel polished instead of plain.

I like the space where logic meets creativity: a useful Python script, a clean UI, a sharp poster, a small automation that saves time, or an AI experiment that teaches me how machines think.

```text
Current mode:
  learn deeply
  build consistently
  design with taste
  improve every week
```

## The Kind Of Projects I Like

I enjoy projects that feel like they have a pulse: not just code that runs, but code that solves a real problem or makes someone say, "wait, that's actually useful."

- **Automation tools** that remove boring repeated work
- **Python apps** for focus, productivity, learning, and everyday systems
- **AI experiments** that help me understand data, models, and intelligent workflows
- **Visual interfaces** that care about layout, color, and feeling
- **Creative builds** where design and programming work together
